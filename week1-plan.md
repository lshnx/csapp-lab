## Python 预告

你学过 C++，学 Python 会非常快。核心区别就一句话：**C++ 什么都要你自己管，Python 什么都帮你管好了。**

| | C++ | Python |
|---|---|---|
| 变量类型 | `int x = 5;` 必须声明类型 | `x = 5` 自动推断 |
| 内存 | `new` / `delete` 手动管理 | 自动回收（垃圾回收） |
| 编译 | 源码 → 编译 → 可执行文件 | 源码直接跑（解释型） |
| 速度 | 快（编译成机器码） | 慢（但有 GPU 库弥补） |

> Python 慢在纯 CPU 计算上，但 PyTorch 的 Python 代码只是指挥棒——底层运算是 C/CUDA 写的，飞快。Python 的角色是"调度"，不是"干苦力"。
