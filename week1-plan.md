---

## 你正在学的东西，到底是为了什么

你现在学的东西是 AI Infra 的**地基**。用一个比喻：

| 你学的东西 | 相当于 | 在 AI Infra 里用来干嘛 |
|------------|--------|------------------------|
| C 语言 | 扳手 | CUDA kernel、底层算子全是用 C/C++ 写的 |
| CS:APP（计算机系统） | 引擎原理课 | GPU 怎么执行指令、cache 怎么影响 kernel 性能、内存怎么分配 |
| PyTorch | 方向盘 | 和 GPU 交互的入口，训练/推理任务全用 PyTorch 写 |
| Linux | 车间 | AI 的所有训练、推理、部署都跑在 Linux 上 |
| Git/GitHub | 工作日志 | 代码版本管理，也是你未来的简历 |

> **你不是在学"编程"，你是在学计算机怎么运转。** 绝大多数搞 AI 的人只会调参，因为他们不懂下面那层。你要做的是懂下面那层的人。

## 本周最重要的一个习惯

```
每学一节 → 写笔记 → git add → git commit → git push
```

笔记写得好不好不重要，写得少也不重要。**每天都 push 才重要。** 到你大三找实习的时候，GitHub 上会有一千多天的 commit 记录，那是你最强的竞争力。
