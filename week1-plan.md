### Linux 每晚 30min 清单

| 天 | 做什么 | 
|------|--------|
| 1 | `ls / cd / pwd / mkdir / touch / rm` — 文件基本操作 |
| 2 | `cp / mv / cat / less / head / tail` — 文件查看和移动 |
| 3 | `man ls` / `man gcc` — 学会看手册，这是 Linux 最重要的习惯 |
| 4 | 配置 git：`git config --global user.name` / `user.email`，生成 ssh key，加到 GitHub |
| 5 | `gcc -Wall -g hello.c` — 编译选项，`-Wall` 开所有警告 |
| 6 | `gdb ./a.out` — 设一个断点，跑过去，打印一个变量 |
| 7 | 把你 csapp-lab 的笔记用终端打开，`cat chapter2-notes.md`，感受在终端里看书的感觉 |
