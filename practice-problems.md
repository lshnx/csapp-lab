## 刷题规则

1. **不挑题，按编号顺序做。** 每题 15-20 分钟，不会就看答案。
2. **每题写一个 C 文件 + `assert` 验证。** 绿色 checkmark 是跑出来的，不是眼睛看的。
3. **做完一道 push 一道。** 每道题一个 commit，别攒一堆。
4. **答案不直接写进练习文件——你自己先想，卡住再问我。**

## 文件命名

```
p{章节}_{题号}.c   例: p2_58.c   p3_06.c
```
